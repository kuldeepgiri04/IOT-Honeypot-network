# Week 3 – Attack Monitoring and Log Analysis

## Objective
To monitor attacker activity and analyze the logs generated by the Cowrie honeypot.

## Tools Used
- Kali Linux
- Cowrie Honeypot
- SSH

## Steps Performed

1. Started Cowrie honeypot.
2. Monitored logs using:
   tail -f var/log/cowrie/cowrie.log

3. Listed recorded sessions:
   ls var/lib/cowrie/tty

4. Viewed attacker session logs:
   cat var/lib/cowrie/tty/<session-id>

5. Analyzed attack data from:
   var/log/cowrie/cowrie.json

## Result
The honeypot successfully captured attacker login attempts and recorded session activity.

## Conclusion
Cowrie honeypot is successfully monitoring SSH attacks and storing attacker interaction logs.

# Week 4 – Attack Analysis

## Observations
During monitoring of the Cowrie honeypot, multiple login attempts were detected.

## Findings

• Multiple brute force login attempts  
• Attackers used common usernames like root, admin  
• Passwords attempted: 123456, admin, root  

## Commands Used by Attackers

• ls
• uname -a
• wget malware.sh

## Conclusion

The honeypot successfully captured malicious login attempts and attacker behavior.  
This shows how attackers try to access IoT devices using weak credentials.
